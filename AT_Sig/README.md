# AT_Sig: 자동매매 시스템 (Quant Trading Engine)

AT_Sig는 키움증권 REST API(혹은 모의투자)를 활용한 **실시간 조건검색 기반 자동매매 프로그램**입니다.
이벤트 구동(Event-Driven) 방식과 강력한 전략 스크립팅 기능을 제공하며, 100% Python으로 작성된 오픈 소스 지향 프로젝트입니다.

## ✨ 주요 기능

*   **실시간 조건검색**: 키움증권 HTS에서 설정한 조건검색식을 실시간으로 포착하여 자동 매매합니다.
*   **유연한 전략 (Python Script)**:
    *   `BackTester` 프로젝트와 호환되는 전략 파일(`New_Strategy.json` 등)을 가져와 사용할 수 있습니다.
    *   단순한 지표뿐 아니라 복잡한 Python 로직(조건문, 변수 등)을 전략에 반영할 수 있습니다.
*   **고성능 & 안정성**:
    *   **DB 캐싱**: 자주 쓰는 데이터는 로컬 DB(`cache.db`)에 저장하여 실행 속도를 극대화했습니다.
    *   **Rate Limiter**: API 호출 제한을 자동으로 준수하여 차단을 방지합니다.
    *   **자동 복구**: 네트워크 끊김이나 장 종료 시 자동으로 대기하거나 재접속을 시도합니다.
*   **텔레그램 제어**:
    *   집 밖에서도 텔레그램 메신저로 `start`, `stop`, `report` 명령을 보내고 계좌 현황을 받아볼 수 있습니다.

## 🚀 설치 및 실행 방법

### 1. 환경 설정
*   **OS 무관**: Windows, Mac, Linux 등 모든 운영체제에서 실행 가능합니다. (REST API 방식)
*   **Python 3.8 이상**: 32bit/64bit 제약 없이 자유롭게 사용 가능합니다.
*   가상환경 (선택):
    ```bash
    conda create -n at_sig python=3.9
    conda activate at_sig
    ```

### 2. 패키지 설치
프로젝트 폴더에서 다음 명령어로 필요한 패키지를 설치하세요.
```bash
pip install -r requirements.txt
```
*(requirements.txt가 없다면: pandas, numpy, requests, matplotlib, pyqt6, websockets 등을 설치)*

### 3. 설정 (Config)
`config.py` 또는 `settings.json`에서 다음 항목을 내 정보에 맞게 수정해야 합니다.
(UI 상에서도 설정 가능)
*   `process_name`: '모의' 또는 '실전'
*   `telegram_token`, `telegram_chat_id`: 텔레그램 알림을 위한 봇 토큰

### 4. 실행
*   **GUI 모드 (권장)**:
    ```bash
    python main.py
    ```
    직관적인 사용자 인터페이스로 전략을 설정하고 매매를 시작할 수 있습니다.

## 📂 프로젝트 구조 (`AT_Sig`)

*   `main.py`: 프로그램 진입점 (Entry Point)
*   `trading_ui.py`: 메인 UI 소스 (PyQt6) — 1,538줄
*   `trading_engine.py`: 매매 핵심 엔진 (이벤트 루프, 상태 관리)
*   `ui/`: **Mixin 모듈** (2026-02-17 분할)
    *   `strategy_mixin.py`: 전략 수식 편집/변환/저장/검증 (469줄)
    *   `account_mixin.py`: 계좌 정보, 일괄 매도, 텔레그램 (320줄)
    *   `settings_mixin.py`: 설정 로드/저장, 조건식 관리 (229줄)
    *   `styles.py`: UI 테마 (Dark Mode QSS)
*   `core/`: 핵심 모듈 폴더
    *   `broker_adapter.py`: 증권사 API 통합 어댑터
    *   `rate_limiter.py`: API 호출 속도 제어
    *   `indicators.py`: 기술적 지표 계산 라이브러리
*   `data_manager.py`: 데이터 수집 및 캐싱 (SQLite)
*   `strategies/`: 전략 파일(.json) 저장 폴더

## ⚠️ 주의사항
*   **실전 투자 위험**: 본 프로그램은 투자를 보조하는 도구일 뿐, 수익을 보장하지 않습니다. 충분히 모의투자로 테스트한 후 실전에 투입하세요.
*   **API 제한**: 키움증권 정책에 따라 조회 횟수 제한이 있습니다. 과도하게 짧은 주기의 조회는 차단될 수 있습니다. (Rate Limiter가 내장되어 있으나 주의 필요)

## 📖 사용 방법 (User Manual)

> [!IMPORTANT]
> **필수 시청 가이드**
> 제3자가 이 프로그램을 처음 사용하는 경우, **[가이드 영상](https://www.youtube.com/watch?v=y4km5VRwV24)** 및 추가 동영상을 시청하시고 이해하신 후, **키움증권 및 텔레그램 관련 작업을 완벽하게 세팅**해야 정상 작동합니다. (개인정보는 각자 신청하여 발급 받아야 합니다)

### 1. 프로그램 시작 및 로그인
1.  `python main.py`를 실행하면 `Sotdanji AutoTrading System` 창이 열립니다.
2.  프로그램이 시작되면 자동으로 로그인 프로세스가 진행됩니다. (모의주식 신청 필수)
3.  좌측 하단의 상태 카드를 통해 **시스템 준비** 및 **웹소켓 연결** 상태를 확인하세요.

### 2. 전략 설정 (필수)
자동매매를 시작하기 전에 어떤 종목을 사고팔지 전략을 세워야 합니다.
1.  **[📋 전략 수립]** 탭으로 이동합니다.
2.  **전략 입력**:
    *   **직접 작성**: 수식 입력창에 전략 수식을 입력합니다. (예: `C > MA(20)`)
    *   **불러오기**: 형제 프로그램인 `Analyzer_Sig`에서 만든 전략 파일(.json)이 있다면, **[전략 가져오기]** 버튼을 통해 쉽게 적용할 수 있습니다.
3.  **적용**: [전략 적용] 버튼을 눌러 설정을 저장합니다.

### 3. 자동매매 시작
1.  좌측 사이드바의 **[▶ 자동매매 시작]** 버튼을 클릭합니다.
2.  **조건검색식 선택**: 키움증권 HTS(`영웅문`)에 저장해둔 조건검색식 목록이 팝업됩니다. 감시할 검색식을 선택하세요. (최대 3개)
3.  이제 프로그램이 실시간으로 종목을 포착하고 지정된 전략에 따라 매매를 수행합니다.

### 4. 텔레그램 알림 설정 (선택)
1.  **[⚙️ 설정]** 탭으로 이동합니다.
2.  텔레그램 봇 토큰과 채팅 ID를 입력하고 저장합니다.
3.  이제 매수/매도 체결 시 스마트폰으로 알림이 전송됩니다.

### 5. 종료
*   **[⏹ 중지]** 버튼을 누르면 매매가 중단됩니다.
*   **[⚠️ 일괄 매도]** 버튼을 누르면 보유 중인 모든 종목을 즉시 시장가로 매도합니다. (긴급 상황용)
