## 1. 프로그램 소개

**AT_Sig**는 키움증권 REST/WebSocket API를 활용한 **완전 자동매매 시스템**입니다.
조건검색식이 종목을 포착하면 전략 필터를 거쳐 자동으로 매수하고, 익절/손절 조건에 맞춰 자동으로 매도합니다.

### 핵심 기능

| 기능 | 설명 |
|------|------|
| **지능형 시장 환경 분석** | **[Update]** BEAR 장세 판정 기준 완화 및 지수 이격도 기반 유연한 대응 |
| **ATR 기반 리스크 관리** | **[NEW]** 터틀 트레이딩의 정수를 이식. 종목 변동성에 따른 다이내믹 손절/익절 및 수량 조절 |
| **스마트 청산(Smart Exit)** | **[Update]** 지수 폭락 및 세력 이탈 감지 시 즉시 전량 매도로 자산 보호 |
| **전략 고도화(Harness)** | **[NEW]** [전고음봉돌파_고도화] 패턴 포착 시 전용 동적 출구 전략(SL/TP) 가동 |
| **전략 우선 원칙(Strategy First)** | **[NEW]** 개별 전략 파일의 타점이 있을 경우, 이를 5번 모드의 초고속 실행 트리거로 자동 승계 |
| **전역 HTS 연동 (V33)** | **[TOTAL NEW]** 더블 클릭 시 **'동기화 스나이퍼 엔진'**이 가동됩니다. 클립보드를 건드리지 않고 메시지를 직접 주입하며, 전송 후 한글 모드를 자동 복구하여 입력 방해를 원천 차단합니다. |
| **전략적 연동(Synergy)** | **[NEW]** Analyzer_Sig가 포착한 주도주를 AT_Sig가 정밀 타격하는 협업 시스템 |
| **텔레그램 알림** | 매수/매도 및 시장 상황 변화를 스마트폰으로 즉시 알림 |

## 4. 매매 방법 선택 가이드 (2026-03-06 업데이트)

> **Sotdanji Lab은 복잡한 설정을 버리고, 직관적이고 강력한 '3대 매매 전략'으로 UI를 전면 개편했습니다.**

### 1️⃣ 조건검색 기반 매매 (Core Strategy)
조건검색식에 포착된 종목을 대상으로 하며, **'전략 선택'** 옵션에 따라 두 가지로 동작합니다.

*   **A. 스피드 모드 (전략 '선택안함' 시)**
    - **동작**: 조건식 포착 즉시 **0.1초 이내**에 시장가 매수를 집행합니다.
    - **추천**: 찰나의 등락이 중요한 초단타 검색식, 이미 검증된 강력한 조건식을 쓸 때.
*   **B. 정밀 검증 모드 (특정 '전략' 선택 시)**
    - **동작**: 포착된 종목의 차트 데이터와 지표를 실시간 분석합니다. 코딩된 전략(예: RSI 과매도 탈출, 이평선 정배열 등)을 통과할 때만 매수합니다.
    - **추천**: 검색식의 허수(속임수)를 거르고 안정적인 타점에서만 진입하고 싶을 때.

### 2️⃣ 시가 갭 회복 (K-Breakout)
*   **동작**: 장 초반 시가 이탈 후, **사용자가 설정한 특정 시간(N분) 동안 음봉 상태를 유지하며 바닥을 다지다가** 다시 시가를 재돌파할 때 매수합니다.
*   **특징**: 단순 흔들기에 당하는 휩소(가짜 반등)를 시간 필터링으로 완벽하게 걸러내어 승률을 극대화했습니다.

### 3️⃣ 관심종목 가속도 공략 (Focus Attack)
*   **동작**: 조건검색 종목뿐만 아니라 사용자가 등록한 **[장전관심종목]** 및 **[Analyzer_Sig 자동전송 종목]** 전체를 레이더로 정밀 추적합니다.
*   **진입 조건**: 단순 가격 상승이 아닌 **체결 가속도 + 매도 총잔량 우위 + 기관 수급 유입**의 3중 필터를 실시간 통과할 때 스나이퍼처럼 진입합니다. (최근 기준 완화로 반응 속도 대폭 개선)

### 🔄 투트랙(Two-Track) 자동 전환 모드
*   **설정**: `use_two_track` 옵션 활성화 시 작동.
*   **동작**: 장 초반(09:00~10:00)에는 **[시가 갭 회복]** 매매에 집중하다가, **10시 정각**이 되면 자동으로 **[관심종목 가속도 공략]** 모드로 전환하여 장중 주도주 공략 체제로 변환합니다.

### 5️⃣ 변동성 마디 공략 (Volatility Node Attack)
*   **동작**: **[NEW]** 종목의 평균 변동성(ATR)을 계산하여, 통계적으로 의미 있는 돌파 지점(`전일종가 + 0.7*ATR`)을 사전에 타점으로 장전합니다.
*   **특징**: 
    - **초고속 집행**: 차트 분석을 매번 수행하지 않고, 미리 정해진 가격에 닿는 순간 0.1초 이내에 즉시 주문을 넣는 **'스나이퍼 모드'**입니다.
    - **전략 우선 원칙**: 특정 전략(예: `BB스퀴즈탈출`)이 선택되어 있다면, 시스템 기본값 대신 **전략이 산출한 정밀 타점을 실행 가격으로 사용**합니다. (정교함과 속도의 결합)

## 6. Analyzer_Sig 연동 (100% 자동화)

**Analyzer_Sig**에서 분석된 **[주도]**, **[과열]**, **[후발]** 타이틀 획득 종목은 별도의 버튼 클릭 없이도 **실시간으로 AT_Sig에 자동 전송**됩니다.
- **수신 즉시 감시**: 전송된 종목은 AT_Sig의 **실시간 시세 레이더(REG)**에 즉시 등록됩니다.
- **가중치 연동**: Analyzer_Sig에서 판정한 테마 순위에 따른 **매수 가중치(Weight)**가 함께 전달되어 매수 비중 결정에 반영됩니다.
- **마스터 관제 연동**: **Master_Control.py(통합 관제)**를 사용하면 Analyzer_Sig의 분석 결과와 매집 분석 데이터가 결합된 `⭐ 황금별(Golden Star)` 신호를 실시간으로 확인하며 더욱 정교한 대응이 가능합니다.

## 7. 고급 기능

### 🔥 장전 선행 로딩 (Warm-up)
HTS 조건식 이름에 **"장전관심"** 또는 **"수동"** 문구를 넣어 저장하면, 해당 종목들의 차트 데이터를 미리 받아둡니다. 장 시작과 동시에 지체 없는 분석 및 주문이 가능해집니다.

### 🛑 지능형 지수 방어 및 자산 비중 조정 (Risk Management)
*   **Market Regime 상태 판단**: 매 10분마다 지수를 분석하여 **BULL / SIDEWAYS / BEAR / CRASH**를 판단합니다.
*   **매매 강도 실시간 최적화 (Multiplier)**: 시장 상황에 따라 한 종목당 투입되는 금액을 자동으로 조절합니다.
    - **BULL (강세장)**: 설정 금액의 **1.2배** 투입 (공격적 매수)
    - **SIDEWAYS (횡보장)**: 설정 금액의 **1.0배** 투입 (표준 매매)
    - **BEAR (약세장)**: 설정 금액의 **0.5배** 투입 (보수적/자산 보호)
*   **폭락장(CRASH) 대응**: 모든 신규 매수를 차단하고, 보유 종목은 **스마트 청산(Smart Exit)** 로직에 의해 즉시 매도하여 현금을 확보합니다.
*   **약세장(BEAR) 대응**: 손절 예약폭을 자동으로 타이트하게 조절(기본 대비 약 +1.5% 상향)하여 리스크를 관리합니다.

### ⚡ 스톡 레이더 (Point 3 - 에너지 폭발 감지)
*   **거래밀도 분석**: 최근 평균 대비 실시간 거래량이 **300% 이상 폭발**하는 결정적 순간을 포착합니다.
*   **호가 에너지**: 매도 총잔량이 매수 잔량보다 많은 **돌파형 호가** 상태를 실시간 점수화합니다.
*   **아이콘 알림**: 에너지가 폭발한 종목은 로그창에 `🔥 [에너지폭발]` 마크와 함께 표시되며, 즉각적인 추격 매수 옵션을 지원합니다.

### 💎 매집 데이터 자동화 및 스마트 그물망 (Accumulation Radar)
*   **스마트 자동 스캔 방어**: 프로그램 시작 시 오늘 자 데이터가 없더라도, API 429 에러(속도 제한) 방지를 위해 **오전 8시 50분 이전에만 150종목 풀 스캔을 자동 수행**하며, 장중이나 임박 시간에는 무거운 데이터 스캔을 안전하게 차단합니다.
*   **그물망 자동 편입 (On-Demand Caching)**: 언제 엔진을 켜든, 이미 로컬 DB(`cache.db`)에 완성된 최상위 매집주 40개를 0.1초 만에 추출하여 웜업 큐와 웹소켓 실시간 구독망에 즉각 밀어 넣습니다.
*   **안전한 비동기 웜업 (Queue)**: 한 번에 40개의 차트를 로딩하더라도, 탑재된 웜업 워커(Worker)가 큐(Queue)를 통해 스스로 통신 간격을 조절하므로 서버 과부하 없이 장전 세팅을 완벽하고 부드럽게 끝마칩니다. (로딩 소요시간 약 20초 내외)

## 8. 자주 묻는 질문 (FAQ)

**Q. 전략을 선택했을 때와 안 했을 때 어떤 차이가 있나요?**
> **'선택안함'**은 조건식 포착 즉시 주문을 나가는 '총알 모드'이고, **전략을 지정**하면 포착되어도 내부 파이썬 로직(지표 등)을 한 번 더 체크하는 '필터 모드'입니다.

**Q. Analyzer_Sig에서 검증한 전략을 사용하고 싶어요.**
> `Analyzer_Sig`에서 완성한 전략 파일(`.json`)을 `shared/strategies/` 폴더에 넣거나, 설정 탭에서 직접 선택하면 AT_Sig에 즉시 반영됩니다.

> **"핵심 로직은 무조건 `shared/` 내부 모듈 하나로 일원화"**

*   **이중 유지보수 구조 탈피**: 기존에 존재하며 코드 수정의 혼란을 유발하던 `Releases/AT_Sig_Release` 와 `[Backups]` 폴더가 **완전 삭제**되었습니다. 이제 개발 및 수정은 오직 `AT_Sig/` 메인 폴더와 `shared/` 통합 디렉토리 위주로 진행됩니다.
