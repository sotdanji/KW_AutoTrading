# BackTester - 솥단지 백테스팅 연구소

키움증권 REST API를 활용한 **고성능 주식 백테스팅 및 전략 최적화 도구**입니다.
사용자가 작성한 키움 수식을 파이썬 코드로 자동 변환하여, 과거 데이터를 기반으로 매매 전략의 유효성을 검증합니다.

> ⚠️ **주의**: 이 프로그램은 **매매용이 아니므로** 키움 **실계좌 모드**로 자동 접속되도록 구성하였습니다. (모의투자 API로는 과거 차트 데이터 조회가 원활하지 않을 수 있습니다)

## 주요 기능

### 1. 📊 강력한 백테스팅 엔진 (Backtesting)
- **수식 변환**: 키움 수식(예: `CrossUp(C, avg(C, 20))`)을 파이썬 코드로 자동 변환
- **유니버스 모드**: 전 종목 대상 검증 지원
- **실시간 로그**: 매매 발생 시 즉각적인 시각적 피드백 제공 (Buy/Sell, 수익률 등)

### 2. 🧬 GA 최적화 (Genetic Algorithm)
- **유전 알고리즘**: 최고의 수익률을 내는 파라미터(이평선 기간, 손절/익절 비율 등)를 자동으로 탐색
- **스마트 검색**: 무작위 대입이 아닌 진화적 기법으로 최적값 도출
- **병렬 처리**: 멀티 코어를 활용한 고속 연산 지원 (기존 대비 약 4.7배 속도 향상)

### 3. 🛡️ 안전성 및 신뢰성
- **문법 검사**: 실행 전 Python 문법 오류 자동 감지
- **데이터 백업**: 중요 데이터 및 소스코드 자동 백업 기능
- **Fail-Safe**: API 오류나 네트워크 단절 시에도 안전하게 종료 및 복구

### 4. 🎨 사용자 친화적 UI
- **다크 테마**: 장시간 분석에도 눈이 편안한 현대적인 디자인
- **직관적 설정**: 복잡한 설정 없이 몇 번의 클릭으로 테스트 시작
- **상세 리포트**: 승률, 손익비, 최대 낙폭(MDD) 등 전문적인 분석 지표 제공

### 1. 설치
터미널(PowerShell)에서 아래 명령어를 실행하여 필수 패키지를 설치합니다.

```powershell

pip install -r requirements.txt
```

`settings.json` 파일이 있는지 확인하고, 필요하다면 `settings.template.json`을 복사하여 생성합니다.



## 🚀 사용 가이드

### 단계 1: 전략 수립 및 변환
1. 프로그램 중앙의 **[📋 전략수립]** 탭을 클릭합니다.
2. **'키움 수식 입력'** 란에 원하는 전략 수식을 입력합니다.
   *(지표변수가 있을 경우, 수식 위쪽에 변수형식으로 입력합니다)*
   - 예시:
     ```
     Period = 20;
     CrossUp(C, avg(C, Period))
     ```
     *(20일 이동평균 골든크로스)*
3. **[🔄 파이썬 코드로 변환]** 버튼을 클릭하여 수식을 파이썬 코드로 자동 변환합니다.
4. 하단 **'변환된 파이썬 코드'** 란에 코드가 생성되었는지 확인합니다.
   - 필요하다면 변환된 코드를 직접 수정할 수도 있습니다.
5. **[✅ 코드 검증]** 버튼을 눌러 문법 오류가 없는지 확인합니다.
6. 전략을 **[💾 저장]** 하여 나중에 다시 사용할 수 있습니다. 필요할 때 **[📂 불러오기]** 로 복원하세요.

### 단계 2: 백테스트 설정 (좌측 패널)
1. **종목 선택**:
   - **전체 종목 (유니버스)**: 입력란을 **비워두면** 전 종목을 대상으로 테스트합니다.
     *(⚠️ API 호출 제한을 고려하여, 기본적으로 상위 **200개 종목**을 샘플링합니다)*
   - **단일 종목**: 특정 종목만 검증하고 싶다면 코드를 입력합니다 (예: `005930`).
2. **조건 설정**:
   - **백테스트 기간**: 시작일과 종료일을 지정합니다.
   - **종목 필터**: 최소 거래금액 및 거래량 조건을 설정하여 대상을 필터링합니다.
   - **실매매 자본**: 초기 예수금과 종목당 투자 비중을 설정합니다.
   - **리스크 관리**: 익절(Take Profit)과 손절(Stop Loss) 퍼센트를 설정합니다.

### 단계 3: 백테스트 실행 및 분석
1. 좌측의 **[Run Backtest]** 버튼을 클릭합니다. *(로그인 상태에서만 활성화됩니다)*
2. 우측 **[실행 로그]** 패널에서 진행 상황을 실시간으로 확인합니다.
3. 완료 시 자동으로 **[📊 성과 요약]** 탭이 열립니다.
   - **수익률, 승률, 누적 수익 차트** 등을 시각적으로 확인합니다.
4. **[💼 거래 내역]** 탭에서 날짜별 상세 매매 기록을 엑셀처럼 확인할 수 있습니다.

### 💡 팁: 전략 최적화 (GA)
**[유전 알고리즘 최적화]** 버튼을 누르면 AI가 전략의 핵심 파라미터를 자동으로 최적화해줍니다.
*(이 버튼은 **전체 종목(유니버스) 모드**에서만 작동하므로, 종목선택 입력란이 비어있는지 확인바랍니다)*

1. **탐색 범위 설정**:
   - **손절 (SL)**: 최소~최대 범위 (예: 1.0% ~ 10.0%)
   - **익절 (TP)**: 최소~최대 범위 (예: 5.0% ~ 30.0%)
   - **비중 (Ratio)**: 종목당 투자 비중 범위 (예: 10% ~ 50%)
2. **GA 엔진 설정**:
   - **개체 수 (Population)**: 한 세대당 테스트할 전략 개수 (추천: 10~20)
   - **세대 수 (Generations)**: 진화 횟수 (추천: 5~10)
3. **[시작]** 버튼을 누르면 최적화가 진행되며, 실시간으로 최고 수익률이 갱신됩니다.
4. 완료 후 **[결과 적용]**을 누르면 가장 좋은 파라미터가 메인 화면에 자동 입력됩니다.
5. **[Run Backtest]** 버튼을 눌러 GA가 추천한 전략을 평가해 봅니다.
