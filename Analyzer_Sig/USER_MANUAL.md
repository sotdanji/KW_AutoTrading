## 1. 프로그램 소개

**Analyzer_Sig**는 실시간 주도주 분석과 전략 검증 기능을 하나로 통합한 고성능 분석 엔진입니다. 
시장의 실시간 데이터를 분석하여 주도 테마와 종목을 발굴하고, 검증된 전략을 시뮬레이션하여 최적의 매매 타이밍을 산출합니다.

---

## 2. 핵심 기능

| 기능 | 설명 |
|------|------|
| **실시간 주도주 발굴** | 거래대금, 등락률, 체결 강도를 종합하여 현재 시장의 대장주를 초단위로 포착 |
| **테마 및 섹터 레이더** | 개별 종목의 상한가/급등 정보를 취합하여 실시간 강세 테마를 분류 (거래대금/거래량 패널 12종목 확장 표시) |
| **전략 백테스팅 엔진** | `shared/strategies/`에 저장된 전략을 과거 600일치 데이터를 기반으로 정밀 시뮬레이션 |
| **매집 분석(Accumulation)** | 주요 창구(외국계/기관)의 입성을 분석하여 세력의 매집 강도를 점수화 |
| **전역 HTS 연동 (V33)** | **[NEW]** 모든 분석 테이블의 종목명 **더블 클릭** 시 '동기화 스나이퍼 엔진' 가동. 한글 모드 자동 복구로 입력 방해 제로 실현 |
| **시장 장세(Regime) 판정** | 지수 이격도 및 추세를 분석하여 BULL/SIDEWAYS/BEAR 상태를 실시간 출력 |

## 3. 실행 방법

1.  **관제 센터(추천)**: `Master_Control.py` 또는 `MASTER_START.bat` 실행 후 'Analyzer_Sig' 카드에서 **[시작]** 클릭.
2.  **단독 실행**: `Analyzer_Sig/Anal_Main.py` 실행.

---

## 4. 주요 모듈 사양

-   **`leader_finder.py`**: 실시간 수급 폭발 종목 및 주도주 포착 로직.
-   **`backtest_engine.py`**: 틱/분/일 기반 다차원 백테스팅 실행기.
-   **`market_engine.py`**: 전체 시장의 건강도 및 지수 연동 데이터 처리.
-   **`theme_radar.py`**: 섹터별 순환매 및 대장주 변경 감지 시스템.
-   **`accumulation_manager.py`**: 7일/20일 누적 창구 데이터 분석 모듈.

---

## 5. AT_Sig 및 Master_Control 연동

-   **데이터 전송**: 분석된 종목 리스트는 `shared/lead_watchlist.json`을 통해 **AT_Sig**로 즉시 전달됩니다.
-   **통합 관제**: **Master_Control** UI에서 Analyzer가 도출한 주도주와 매집 점수를 실시간으로 모니터링할 수 있습니다.
-   **황금별(Golden Star)**: 주도주 신호와 고득점 매집 데이터가 일치하는 종목은 '관심 종목'으로 자동 격상됩니다.

## 🚦 중요 운용 주의사항 (Critical Notes)

1.  **API 키 충돌 방지**: 
    - `AT_Sig`와 `Analyzer_Sig` 등 여러 프로그램을 동시 실행할 경우, **동일한 모의투자 계좌 키**를 중복 사용하면 API 세션이 끊기거나 데이터 조회가 제한될 수 있습니다.
    - 가능한 각 프로그램마다 다른 계좌 키(종합1, 종합2, 캐치 등)를 배정하여 운용하십시오.
2.  **데이터 무결성**:
    - 백테스트 및 분석 데이터는 `data/` 폴더 내의 SQLite DB를 참조합니다. 관제 센터(`Master_Control`)가 실행 중일 때 DB 파일을 강제로 삭제하지 마십시오.
3.  **전략 관리**:
    - 모든 매매 전략은 `shared/strategies/` 내의 `.json` 파일로 관리됩니다. 파일명 수정 시 타 프로그램과의 연동 여부를 반드시 확인하십시오.
